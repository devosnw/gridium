# Gridium

### Prerequisites

* python3
* pip

### Installation

```sh
git clone git@github.com:devosnw/gridium.git
cd gridium/

# not required but recommended to do inside a virtual environment
mkdir venv/
python3 -m venv venv/
source venv/bin/activate

make install
```

### Tests

```sh
make test
```

### Run server

```sh
honcho start
```
